# PersonalPortfolio_OJ
 Personal Portfolio

Kjo portfolio eshte krijuar duke perdorur teknologjite HTML, CSS, JavaScript <br />
per pjesen e Front-end dhe PHP/MySql per pjesen e Back-end. Eshte responsive. <br />
PHP se bashku me bazen e te dhenave(file i bazes se te dhenave mund te gjendet <br/>
ne folderin database files) jane perdorur per t'i shtuar keto funksione: <br/>
 1. Per t'i bere fetch disa te dhena nga baza e te dhenave dhe per t'i shfaqur ato <br/>
    ne UI <br/>
 2. Per ta funksionalizuar pjesen e kontaktit (dergimin e emails) <br/>
 3. Per ta funksionalizuar butonin per shkarkimin e CV <br/>
 
 Ne folderin demo mund te gjendet video si demo e projektit (nuk mund te hostohej online </br>
 per shkak se ka shume kod).
